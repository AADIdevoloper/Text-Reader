## Getting Started



### Prerequisites

Python
Pyttsx3
PyPDF2

### Installation

1. Install python from https://www.python.org/downloads/
2. Install pyttsx3
   ```sh
   pip install pyttsx3
   ```
3. Install PyPDF2
   ```sh
   pip install PyPDF2
   ```
